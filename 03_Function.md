# Kotlin : Functions

- In kotlin, functions are first class concepts.
- We can write and use functions without any kind of enclosing class, struct or other obstruction.

```kotlin
fun getGreeting(): String {
    return "hello kotlin"
}

fun main() {
    println(getGreeting())
}
```
